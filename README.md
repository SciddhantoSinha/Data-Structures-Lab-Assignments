# Data Structures Laboratory (C++)

## **Overview**
This repository contains **C++ programs** implemented as part of the **Data Structures Laboratory (S.Y. CSE)**.
The experiments focus on understanding **fundamental data structures**, **algorithms**, and their **practical applications**.

All programs are written in **C++** and compiled using the **g++ compiler**.

## **List of Experiments**

### **Experiment 1: Matrix Operations**
- **Matrix Addition**
- **Matrix Subtraction**
- **Matrix Multiplication**
- **Time and Space Complexity Analysis**

### **Experiment 2: Saddle Point in Matrix**
- Finding an element that is **minimum in its row** and **maximum in its column**

### **Experiment 3: Random Password Generation**
- Generation of all possible passwords using **nested loops**

### **Experiment 4: Matrix Storage Operations**
- **Upper Triangular Matrix** check
- **Sum of diagonal elements**
- **Transpose of matrix**

### **Experiment 5: String Operations**
- **Character frequency**
- **Delete characters by position**
- **Delete a specific character**
- **Palindrome check**

### **Experiment 6: Polynomial Operations Using Structure**
- **Polynomial representation**
- **Polynomial addition**
- **Polynomial multiplication**

### **Experiment 7: Singly Linked List (Pinnacle Club)**
- **Add and delete members**
- **Count total members**
- **Display members**
- **Reverse display of list**

### **Experiment 8: Infix to Postfix Conversion Using Stack**
- **Infix to postfix conversion**
- **Postfix expression evaluation**

### **Experiment 9: Job Queue Simulation (Priority Queue)**
- **Queue operations using array**
- **Priority-based job scheduling**

### **Experiment 10: Searching Techniques**
- **Linear Search**
- **Binary Search**
- **Object-Oriented implementation**

### **Experiment 11: Sorting Techniques**
- **Selection Sort**
- **Bubble Sort**
- **Insertion Sort**

## **Technologies Used**
- **Language:** C++
- **Compiler:** g++
- **Concepts:** Arrays, Strings, Structures, Stack, Queue, Linked List, Searching, Sorting

## **How to Run**
g++ filename.cpp -o output  
./output
